Featured Projects

🔭 Reward Hacking Observatory

Empirical AI-safety testbed for agentic reward hacking

A controlled experimental environment that empirically measures how LLM agents game naive reward signals. 30 tasks across 3 ambiguity levels, 16 MCP tools over a FastAPI + SQLite workspace, three reward functions (naive → intermediate → robust with ground-truth state verification), and a real-time detector for 5 hack types — revealing that reward hacking is ambiguity-driven and systematic.

51.7% overall hack rate detected 0% → 80% hacking as ambiguity rises 195+ tests incl. regression locks
MCP FastAPI Anthropic SDK Reward Design AI Alignment

🎯 AgentRubric

Multi-agent RLHF rubric optimization & reward-model training

A LangGraph 7-node multi-agent system (transcript filter → gold judge → rubric designer → reward scorer → rubric critic → hack detector → preference store) that auto-generates preference pairs for reward-model training, detects proxy-vs-gold judge divergence to catch reward hacking, and fine-tunes Qwen2-1.5B via QLoRA on self-generated preference data — all on a 4GB-VRAM consumer GPU.

97.5% preference-pair success rate 33.5% MAE reduction vs baseline Trained in 347s on RTX 3050
